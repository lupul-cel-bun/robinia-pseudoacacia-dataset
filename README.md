# robinia-pseudoacacia-dataset
## Measurements on a pot with black locust saplings

*Time* - ISO 8601 compiant, timezone is GMT+2 <br>
*Weight* - weight of pot, potting mix, saplings, seeds and whatever water.<br>
*Stems* - number of living saplings with true leaves<br>

Pot is outside, facing west, tag towards the east.
